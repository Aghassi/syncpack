# syncpack 

Normalise differences in packages/*/package.json files in Lerna Monorepos

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install syncpack --save
```


## Tests

```sh
npm install
npm test
```
